# GraphicsRayTracing
Ray traceing project C++ Visual Studio

## Implemented
Shapes:
  * Sphere
  * Cylinder
  * Flat
  * Torus

Surfaces/Materials:
  * Glass
  * Gold
  * Silver
  * Smooth
  * Rough
